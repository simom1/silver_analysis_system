## 📝 使用建议

### 对于用户
1. **首次使用前**
   - 复制配置模板文件
   - 填入自己的真实配置
   - 确保配置文件在.gitignore中

2. **日常使用**
   - 定期运行安全检查
   - 避免在代码中硬编码敏感信息
   - 使用环境变量存储配置

3. **上传代码前**
   - 运行 `敏感信息检查.bat`
   - 使用 `安全上传.bat` 进行上传
   - 检查Git diff确认无敏感信息

### 对于开发者
1. **代码开发**
   ```python
   # ❌ 错误做法
   API_KEY = "real_api_key_here"
   
   # ✅ 正确做法
   API_KEY = os.getenv('API_KEY', 'your_api_key_here')
   ```

2. **配置管理**
   ```python
   # 使用配置文件模板
   config_template = {
       "mt5_login": "YOUR_MT5_LOGIN",
       "mt5_password": "YOUR_MT5_PASSWORD"
   }
   ```
